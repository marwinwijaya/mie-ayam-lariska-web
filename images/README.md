## Upload Workflow

1. Login ke admin dashboard
2. Edit menu item
3. Upload foto di form
4. Klik "Download Foto (Rename Otomatis)"
5. Taruh file yang di-download ke folder ini
6. Git commit dan push

## Format yang didukung

- JPG/JPEG (recommended)
- PNG
- WebP

## Ukuran maksimal

- 2MB per file
- Rekomendasi: 400x300px atau 4:3 ratio
